# Notium
## RU
Notium – браузерное расширение для заметок прямо на любых сайтах. Сохраняйте идеи, ссылки и важную информацию для каждой страницы. Открывайте заметки вручную, выбирайте цвет и управляйте всеми заметками через удобный менеджер заметок.
## EN
Notium – a browser extension for taking notes directly on any website. Save ideas, links, and important information for each page. Open notes manually, choose a color, and manage all your notes through a convenient notes manager.
